第一单元 · 风险管理与保险▶

1.1 风险概述

风险的定义（四种视角）

  • 传统定义：结果的不确定性，通常伴随损失的可能性
  • ISO标准：不确定性对目标的影响
  • 损失可能性：风险是损失发生的可能性，或可能发生的损失
  • 偏离期望：风险是结果对期望（目标）的偏离，损失的变化范围越大风险越大（即使期望损失相同）

风险的分类

  • 可分散风险（非系统性风险） vs 不可分散风险（系统性风险）
  • 纯粹风险(Pure Risk)：只产生损失无收益 —— 保险只保纯粹风险！（所以市面上是医疗险、寿险、财产险，而非股票保险、期货保险）
  • 投机风险(Speculative Risk)：收益和损失皆有可能（如炒股、赌博）

纯粹风险的六大特性（核心考点）

  • 客观性：客观存在无法消除
  • 偶然性：是否发生、发生时间、地点、程度等不确定（必然发生的事不能称之为风险）
  • 普遍性：无处不在，无时不有
  • 社会性：与人类社会利益相关（暴风雪发生在南极北极就不算风险）
  • 可测性：可计算发生概率及损失大小（通过大数定律）
  • 可变性：性质、数量及大小会发生变化

1.2 风险构成要素（因果关系链）

风险因素(Hazard) → 风险事故(Peril) → 损失(Loss)

  • 有形风险因素(Physical Hazard)：导致损失发生的物质因素（如路面结冰、设备老化）
  • 无形风险因素(Invisible Hazard)：
    • 道德风险因素(Moral Hazard)：不诚实、不良企图或欺诈行为（怀有不良企图）
    • 心理风险因素(Morale Hazard)：行为上的粗心大意和漠不关心（态度风险因素）
  • 风险事故(Peril)：导致损失发生的直接原因——自然现象（天灾）、社会政治经济变动（人祸）、人或物内在属性缺陷（疾病、设备故障）
  • 损失(Loss)：偶然的、非预期的、非计划的经济价值减少或灭失，必须可用货币计量
    • 直接损失：风险事故对标的本身造成的破坏事实
    • 间接损失（关联损失）：由于直接损失所引起的破坏事实——损失的是未来可得利益，必须具有实际意义且属于直接影响所及范围（厂房机器损毁导致营业中断的利润损失要赔；因堵车迟到被扣奖金不赔——相关性不够紧密）

注意：同一事件在不同情况下可能是风险因素也可能是风险事故——暴雪导致房屋被压垮→暴雪是风险事故（气候变化是风险因素）；暴雪导致汽车打滑撞垮房屋→暴雪是风险因素（车祸是风险事故）

1.3 风险种类（多维度分类）

  • 按起因：自然风险（地震洪水台风）vs 社会风险（偷盗抢劫战争）
  • 按发生形态：静态风险（自然不规则运动+人的过错行为）vs 动态风险（社会经济政治变动）
  • 按损失后果：财产风险（物理损坏/贬值/征收）、人身风险（健康&生命）、责任风险（过失侵权→交强险/产品责任/公众责任注意只有责任限额）、信用风险（经济交往违约→信用保证保险）
  • 个人风险（个人和家庭）：失业、早逝、退休后收入不足、健康衰退、财产损失、责任
  • 商业风险（企业）：财产、责任、间接损失（种类少但复杂程度更高）

1.4 风险管理

风险管理概念：识别和度量风险，采用合理手段处理风险，以最小成本保障最大利益。主体是独立的经济单位，强调人的主动行为。

风险成本 = 实际成本 + 无形成本 + 预防控制成本

  • 实际成本：风险事故发生的直接损失+间接损失
  • 无形成本：无法准确衡量价值的损失（名誉损失、机会成本损失）
  • 预防控制成本：为预防控制风险采取措施所花费的费用（安装报警设备、购买灭火器材等）

风险管理过程（六步）：确定风险管理目标 → 识别风险 → 衡量风险 → 处理风险 → 评估风险管理绩效 → 调整风险管理方案

风险管理目标：损失前目标（降低概率+降低损失）；损失后目标（降低直接和间接损失）

风险识别方法：专家法、清单法、财务报表分析、流程分析、投入产出分析、现场检查、历史统计分析

风险衡量（Heinrich三角图 1:30:300）：1次最大伤害事故 ← 30次小伤害事故 ← 300次无伤害事故

  • 损失概率（损失机会）
  • 损失程度：发生损失金额的算术平均
  • 损失变异性：损失的波动程度

风险处理两大路径

  • 风险控制（控制概率和程度）：风险回避 → 损失控制（预防降低概率+抑制减少幅度）→ 风险单位隔离（分离或复制）
  • 风险融资（资金转移）：保险（不消除风险，只转移）→ 非保险转移（出售/出租）→ 风险自留/自保(Self-insurance)

效益成本比：大于1时该策略可取；多个方案比选最大值

1.5 可保风险(Insurable Risk)

传统五大判定标准

  • 纯粹风险：只产生损失无收益
  • 损失可测性：必须可以通过数学方法测算发生概率（需要大量过往案例）
  • 损失金额明确性：损失须可用货币计量（责任险→责任限额；寿险→生命无价定额给付）
  • 损失偶然性：发生与否、时间、程度不可控
  • 损失经济可行性：保费水平可被接受（航天/卫星发射可采用多家保险公司联合承保）

传统标准遇到的挑战 → 不可保→可保的核心因素

  • 人类需求与社会稳定（美国2002年《恐怖主义风险保险法案》要求保险公司将恐怖袭击列为可保；2019年新西兰基督城枪击案——保险公司虽将恐怖袭击列为不可保但全部理赔）
  • 保险公司责任感和品牌荣誉感
  • 技术进步（科技水平发展将不可保风险转化为可保风险，如建立互联网大数据库预测地震风险）

承保不可保风险的具体方法：联合保险、再保险、金融工具应用、现代化科技手段

经典案例（5个）

1. 张三家庭风险案例分析：30岁自由职业者，单身，与年迈父母同住，自住房（贷款未还清）+汽车。风险识别：房屋及家庭财产损失风险、车辆（被盗/损坏/第三者责任/乘客安全）风险、本人和父母健康意外风险、收入中断风险（张三为家庭唯一收入来源）。管理方案：综合人身保险（重大疾病险+寿险）+ 家财险 + 车险（含车上人员保障，因父母也要坐车）。

2. 企业火灾期望值决策（核心计算题）：建筑物价值1000万（可保价值750万，已扣除土地地基250万），火灾必导致全损，间接损失280万。火灾概率5%，安装预防设备后概率降至3%，预防设备价值10万（遇火灾全损），保费6万。

方案一（风险自留）：E = 1030×5% + 0×95% = 51.5万

方案二（自留+风险控制）：可保损失降至500万，间接损失降至140万。E = 650×3% + 10×97% = 29.2万。效益成本比 = (51.5-29.2)/10 = 2.23

方案三（购买保险）：E = 286×5% + 6×95% = 20万。效益成本比 = (51.5-20)/6 = 5.25 → 最优方案

3. 酒后驾车险：天安保险2003年推出"非常事故损失特约险"，作为三者险附加险。满足传统五项可保标准但引发巨大争议——纵容酒驾vs以人为本保障第三方。网上调查大部分人认为会纵容酒驾。事实：一年仅售约300份，无一理赔，次年停售。启示：社会、心理和文化因素对可保性的影响——符合传统标准不一定可保，保险具有社会性，需符合社会需求。

4. 地震巨灾保险：一般财产险将地震列为除外责任。满足传统五项标准但经济可行性存疑（巨灾损失巨大，单个保险公司难以承受）。应对：巨灾债券转移风险、政府支持、科技水平提高（风险数据库建立），2016年首个国家倡议的地震保险产品面世。

5. 恐怖袭击保险：早期可保→后列为除外责任→需求变化后重新可保。美伊战争期间，虽然费率大大提高但仍在承保。核心驱动因素：人们的特定需求、保险公司社会责任、技术进步。但随着局势变化（如通过霍尔木兹海峡被认为必然受损），可能再次不可保——体现可保性的动态变化。

第二单元 · 保险概述▶

2.1 保险的起源与发展

西方古代保险雏形

  • 古埃及（公元前4500年）：石匠互助基金组织——收缴会费支付会员死后丧葬费用（最早人寿/意外险雏形，待考证）
  • 古巴比伦（公元前2250年）：汉莫拉比法典中有类似火灾保险规定，国王命令向居民征税筹集火灾及天灾救济基金
  • 古罗马：年老体弱者聚集共同出资抵御疾病、灾难和死亡
  • 地中海：共同海损分摊原则载入《罗地安海商法》（公元前916年）——"一人为众，众为一人"
  • 中世纪欧洲：基尔特(Guild)制度——职业相同者相互扶助，对会员死亡、火灾、疾病、盗窃等互助共济

中国古代保险思想

  • 3000年前"分舟运米"；西周后备仓储；春秋仓储制度；西汉"常平仓"制度；隋文帝"义仓"+"官仓"
  • 镖局：押运货物和安全保障——原始保险形式。乾隆年间山西人神拳张黑五创办兴隆镖局

现代保险里程碑（按时间顺序）

  • 1347年：第一张商业保单——意大利商船"圣科勒拉"号（现代商业保险起源：海上保险）
  • 1583年：最早人寿保单——伦敦理查德·马丁
  • 1666年：伦敦大火（燃烧五天五夜，85%被烧毁）→ 1667年牙医巴蓬创办火灾保险
  • 1662年：John Graunt制作第一张生命表；1693年哈雷制作人口死亡率表（用于预测死亡率）
  • 1762年：第一家现代人身保险公司——英国伦敦公平人寿保险社
  • 17世纪中叶：最早再保险（海上再保险）；1813年最早火灾再保险合同
  • 1855年：最早责任保险——英国铁路承运人责任险
  • 1895年：最早汽车保险——英国汽车第三者责任险
  • 1911年：最早航空保险——英国劳合社

新中国保险业

  • 1949年10月20日：中国人民保险公司成立
  • 1958-1980年：国内保险业务停办
  • 1980年：恢复；1998年10月撤销人保

2.2 保险的概念

保险五大基本特征（核心考点）

  • 风险转移：主体是投保人（将风险转移给保险人）
  • 风险分散：主体是保险人（聚合同质性标的，大数定律）
  • 基金聚积：收取保费建立保险基金
  • 损失分摊：事后，基于大数定律，出险者和未出险者共同分摊成本
  • 损失补偿：对约定风险导致的损失提供经济保障

风险分散 vs 损失分摊：都基于大数定律，聚焦大量同分布相互独立样本。风险分散是事前的，损失分摊是事后的。

损失分摊数学推导（重点理解）

  • 独立同分布：甲乙两人各拥有价值20万的车，全损概率10%。个人承担：期望损失=2万，方差=36。两人分摊后：期望不变=2万，方差=18（降一半）。n→∞时方差→0，风险趋近于零
  • 不独立但同分布：协方差不为零 → 无法通过增加人数降低风险（一个人的损失会影响其他人）
  • 独立但不同分布：破车和豪车、营运车和家用车分开 → 方差更大，放大了风险
  • 结论：只有独立同分布时，才能准确用大数定律实现风险分散并进一步损失分摊

保险定义（保险法第2条）：投保人支付保费，保险人对约定事故承担赔偿保险金责任（财产险）或给付保险金责任（人身险）。
社会保险定义：国家通过立法建立社会保险基金，对劳动者年老、疾病、伤残、失业、生育等给予物质帮助的社会保障制度。

保险对象（保险标的）

  • 人身标的物：被保险人的生命和身体机能
  • 非人身标的物：标的物的经济价值（有形：房屋车辆货物家畜；无形：信用责任债权）
  • 区分标准：能否准确估价、能否复原、能否转让

主要参与者：投保人（缴费）→ 保险人（承保）→ 被保险人（财产/人身受保障）→ 受益人（仅人身险有，被保险人死亡时享保险金请求权）

保险 vs 赌博

  • 赌博产生投机风险 vs 保险处理纯粹风险；赌博可能获利 vs 保险无获利可能
  • 赌博可针对与赌徒无关的事 vs 保险强调可保利益原则；赌博无社会利益 vs 保险对社会有益

保险 vs 对冲：转移风险不同；补偿原理不同（大数定律 vs 反向操作）。最相似的金融衍生品：期权(Option)、信用违约互换(CDS)。

自保(Self-insurance)：风险自留形式。优点：无信息不对称、避免逆选择/道德风险、节约成本。缺点：规模经济劣势、风险分散不充分、占用母公司成本、风险管理弱于商业保险公司。我国自保公司：中海油、中石化、中石油、中广核、中铁、中远海运、广东粤电、上海电气。

2.3 保险的分类

  • 按性质：商业保险（营利/自愿）、社会保险（非营利/强制/社会公平）、政策保险（政府政策目的：农业/信用/输出/巨灾）
  • 按标的：人身保险、财产保险、责任保险、信用保证保险
  • 按实施方式：强制保险（交强险——商业保险中唯一强制）、自愿保险
  • 按危险转移层次：原保险/再保险；复合保险（投保人投保多家但保额未超实际价值）vs 重复保险（保额超实际价值，不允许）vs 共同保险（保险公司主动联合共同承保）
  • 定值保险：双方事先约定保险价值（古董/艺术品等价值难以确定的标的）。赔偿额 = 保险金额 × 损失程度；损失程度 = (实际价值-残值)/保险标的实际价值。举例：古董投保时确定价值5千万，运输中发生损失仍按5千万为基准按比例赔偿。
  • 不定值保险：按损失时实际价值确定。保障程度(≤1) = 保险金额/损失时完好实际价值；赔偿额 = 损失额 × 保障程度
  • 不足额投保产生原因：少缴保费、保险人只接受不足额投保（风险高）、合同签订后标的价值上涨。赔偿金 = 损失金额 × (约定保险金额/保险价值)

保险法第55条：约定保险价值的按约定价值计算（定值）；未约定的按事故发生时实际价值计算（不定值）；保险金额不得超过保险价值，超过部分无效应退保费；不足额保险按保额与保险价值比例赔偿。

定值保险例题：约定标的价值100万保额100万，出险时实际价值(1)90万→全损赔100万/部分50%赔50万；(2)110万→全损赔100万/部分50%赔50万。
不定值保险例题：保额100万，出险时实际价值(1)90万→全损赔90万/部分50%赔45万；(2)110万→全损赔100万/部分50%赔50万（定额限制在保额内）。

2.4 保险的代价

  • 运营成本：纯保费 + 成本支出 = 毛保费
  • 逆向选择（事前信息不对称）：高风险人群更倾向投保→信号传递（投保人主动告知）+ 信号筛选（保险人主动核保）→ 最大诚信原则
  • 道德风险（事后信息不对称）：投保后行为改变增加损失→设置免赔额 + 加强监督（但免赔额客观上刺激夸大信息，矛盾）

经典案例（3个骗保案）

1. 2020年福建骗保案：保险代理人甲伙同客户乙等5人，虚构家庭成员、伪造投保资料为已故人员购买重疾险。两年后以被保险人病故为由申请理赔并提交虚假材料。最终6人因保险诈骗罪获刑，挽回损失89万元。涉及保险法27条（谎报事故/编造虚假原因）。

2. 2019年邵某断指骗保案：邵某先后在多家保险公司投保人身意外险900余万元。自称在家剁大骨头不慎砍掉左手拇指食指。疑点：短时间多份投保、断指均至根部拒不接植、配偶2019年初因类似意外获赔23万。法院判保险诈骗罪（未遂），有期徒刑三年缓刑四年，罚金2万元。体现逆选择风险+道德风险。

3. 重庆父子团伙骗保案：张某驾车轧坏王某手包（包内86万名表+手机），理赔成功。同年4月重庆又发类似案件——两起肇事司机为父子关系。深入调查发现团伙骗保，网上购买二手名表制造假事故。体现道德风险——事后信息不对称。

第三单元 · 保险基本原则（重点）▶

3.1 最大诚信原则(Utmost Good Faith)

起源：1766年Carter v Boehm案，Mansfield勋爵判罚；最早成文法：1906年《英国海上保险法》。我国保险法第5条："保险活动当事人行使权利、履行义务应当遵循诚实信用原则。"

含义：充分、准确、双向告知有关保险的所有重要事实（是否影响缔约）；不允许欺骗（提供虚假信息）和隐瞒（该提供的信息故意不予提供）。

规定原因

  • 保险活动中的信息不对称：投保人对保险标的信息优势 + 保险人对合同条款的信息优势（附和性合同由保险人制定）
  • 保险本身的不确定性：风险事故发生具有不确定性

告知（如实告知）

  • 时间：合同订立前/订立时/订立后均有告知义务（如以家用车名义投保后转去开滴滴→须告知）
  • 投保人告知：订立时如实填写投保单并签字；合同有效期内危险程度增加须告知；标的转移须通知保险人确认；事故发生后及时通知；有重复保险须通知
  • 投保人告知形式：主观告知（询问回答告知）——仅对保险人提出的问题如实回答（现行主流）；客观告知（无限告知）——与标的风险状况相关的内容无明确规定全部告知
  • 保险人告知：订立时说明合同条款内容（特别是免责条款）；约定条件满足时履行赔偿给付义务
  • 保险人告知形式：明确列示（合同中告知）+ 明确说明（对投保人解释，特别是免责条款）

保证(Warranty)

  • 对某一事项的作为/不作为、某状态存在/不存在的许诺（如盗窃险要求仓库24小时专人看管）——目的是控制风险
  • 违反保证→保险人可宣告合同无效！无需证明因果关系
  • 明示保证：认定事项保证（确认保证，涉及过去与现在）+ 约定事项保证（承诺保证，涉及现在与未来）
  • 默示保证（海上保险）：适航性、不绕行（按预定航线）、合法性（不非法经营/不运输违禁品）

弃权与禁止反言（重点）

  • 弃权：放弃保险合同本可主张的权利
  • 禁止反言：不得要求重新主张已经放弃的权利
  • 不可抗辩条款（保险法第16条）：合同成立超过2年+被保险人生存→保险人不得解除合同（即便不知道也要赔付）；自保险人知道解除事由起超30日不行使而消灭
  • 视为保险人弃权的五种情况：明知违背义务仍接受补缴保费；明知有拒赔抗辩权仍要求提供损失证明；明知损失证明有纰漏仍无条件接受；事故后投保人逾期通知仍接受；明知违背义务仍保持沉默
  • 弃权与禁止反言一般在人寿保险中出现（财产险多一年期，不涉及），时间限制2年

违反最大诚信原则的法律后果

  • 投保人故意：不赔不退
  • 投保人重大过失：不赔但退保费
  • 保险人在订立时已知投保人未如实告知：不得解除合同，须赔付
  • 保险人违反告知（免责条款未明确说明）：该条款不产生效力
  • 有利被保险人解释（保险法第30条）：对合同条款有争议时，法院或仲裁机关应作有利于被保险人和受益人的解释
  • 保险人违反告知的行政处罚（保险法第139条）：阻碍告知/诱导不告知/非法回扣→对保险公司5-30万罚款，对人员2-10万罚款
  • 司法解释（二）第6条：投保人告知义务限于保险人询问范围；保险人以概括性条款拒赔不被支持，须具体列举
  • 违反保证：保险人有权解除或不赔付且不退保费。例外：环境变化无法履行（海上保险）、法律法规变更、保险人事先弃权

3.2 保险利益原则(Insurable Interest)

定义（保险法第12条，考试必考）：投保人或被保险人对保险标的具有的法律上承认的利益。性质：①保险合同的客体（非保险标的本身）②合同生效的依据（无保险利益→合同无效）③不等于保险合同利益（保险金请求权）。条件：合法、客观存在、确定的经济利益。

财产险保险利益：所有权（完全利益）、使用权、经营权、承运权/保管权、债权（如银行对抵押品商品房）、潜在责任（如洗衣店对客户衣服）

  • 包括：现有利益、预期利益（租金）、责任利益（车辆三者/产品/雇主）、合同利益（国际贸易）

人身险保险利益（保险法第31条）：本人；配偶、子女、父母；有抚养/赡养/扶养关系的家庭成员/近亲属；有劳动关系的劳动者（受益人必须是其亲属，不能是公司法人）；被保险人同意投保（视为具有保险利益）

适用时限（核心考点！）

  • 人身险：合同订立时必须具有保险利益
  • 财产险：保险事故发生时必须具有保险利益

坚持保险利益原则的意义：防止赌博行为、防止道德风险、明确保险责任、限制保险金额（规定保障最高限度，不允许额外获利）

保险利益的变动：财产险——标的灭失→保险利益灭失；继承/让与/破产→保险利益转移。人身险——被保险人自杀等除外责任原因死亡→保险利益灭失；债权债务关系→可转移，由继承人继承；亲属关系→无法转移。

保险法第49条（财产险标的转让）：受让人承继被保险人的权利义务；须及时通知保险人（货物运输险除外）；危险程度显著增加→30日内保险人可增保费或解除；未通知→因转让导致危险显著增加发生的事故不赔。

3.3 近因原则(Proximate Cause)

含义：引起保险标的损失最直接、最有效、起决定作用的因素——效果上最接近，非时间空间上最接近。如近因属于保险责任→赔付；如近因属于除外责任→不赔付。

近因认定的四种情况

  • 单一原因致损：直接认定
  • 多种原因同时致损：看是否同属承保/除外责任；若皆有→损失可明确界定→分别处理；损失无法明确界定且无法确定近因→按比例（司法解释三第25条）
  • 多种原因连续发生：追溯最根本原因
  • 多种原因间断发生：看是否插入新的独立原因

3.4 损失补偿原则（重点）

含义：以保险责任范围内的损失发生为前提，以被保险人的实际损失为限，不能因损失获额外利益。金额限制：免赔额 + 赔偿限额 + 保险利益。

主要体现：财产险中的不定值保险 + 人身险中的医疗费用补偿。

意义：约定赔偿准则（足额/不足额/超额投保的赔偿标准）、防止额外获益、防范道德危险

例外（不适用损失补偿原则）：定值保险（古董/艺术品/名画/传家宝）、重置成本保险（赔付不扣除折旧）、人寿保险（生命无价，约定给付）、重大疾病保险（定额给付）

重复保险分摊（保险法第56条）

  • 比例分摊：赔偿金额 = 损失金额 × 该保险人承保保额/所有保险人承保保额总和
  • 限额分摊：赔偿金额 = 损失金额 × 该保险人赔偿限额/所有保险人赔偿限额总和
  • 顺序分摊：按承保顺序依次赔付
  • 例题：房屋50万，甲保30万乙保50万，损失40万。比例分摊：甲40×(30/80)=15万/乙25万。限额分摊：甲40×(30/70)=22.86万/乙40×(40/70)=17.14万。

代位追偿（权利代位，保险法第60条）：保险人赔偿后在赔偿金额范围内代位行使被保险人对第三者请求赔偿的权利。

  • 条件：对保险人和第三者同时存在赔偿请求权 + 要求第三者赔偿 + 保险人须先履行赔偿责任方可代位
  • 金额：以已赔付金额为限；被保险人获赔小于损失→有权继续向第三者追偿
  • 适用范围：财产保险+涉及医疗费用的健康险；人身险给付性（死亡/伤残）不适用（保险法第46条）
  • 不得对被保险人及其家人/组成人员行使代位追偿权（不被视为第三者）

委付（物上代位，保险法第59条）：发生保险事故造成标的推定全损→将标的全部权益转移给保险人→请求按保险金额全数赔偿（常见于海上保险）。

  • 条件：推定全损；由被保险人向保险人提出；就整体标的提出要求；需经保险人同意（可拒绝）；不得附加条件
  • 与代位追偿区别：代位追偿是权利代位，保险人无须承担其他义务；委付是物上代位，保险人一并接收标的权利和义务。代位追偿保险人不可拒绝，委付保险人可拒绝。

经典案例（15个，本章考试重点最多）

1. Carter v Boehm (1766) —— 最大诚信原则起源：马尔堡总督Carter投保海上保险，未告知堡垒防御缺陷。法国攻克马尔堡后，保险公司以未如实披露为由拒赔。Mansfield勋爵判：保险人明知风险存在却未主动询问而承保，不能证明投保人故意隐瞒；投保时战争仅是可能性，位于伦敦的保险公司更应了解战争风险（可视为公共信息）；认定保险公司履行赔偿责任。确立最大诚信原则。

2. Notre Dame船保证条款案：船从利物浦启航时46名船员（保证条款要求≥50），途中补足至50人后被法国人捕获。保险公司以违反保证条款拒赔——法院支持：违反保证条款即可宣告合同无效，无需证明因果关系。

3. Stipcich v Metropolitan Life (1928)：投保人5月投保答"无胃病"→6月确诊胃溃疡→7月1日保单生效→12月因胃癌去世。法院判保险公司无需赔付：保险合同订立前、订立时、订立后均有告知义务。若保单签发、合同生效之后才诊断出来则正常赔付。

4. 张三为母亲投保高血压案：母亲有高血压病史但病例显示系孙子患病导致上火、短期血压增高、医嘱适当用药。法院认定系短期非长期疾病，保险公司需理赔。保险人签合同时有权调取医保档案，应自行核查。

5. 小李慢性肝炎案——超过两年不可抗辩：小李2015年4月投保人寿险附加重疾未告知慢性肝炎，2018年2月心肌梗塞理赔。因合同超过2年，适用不可抗辩条款，保险公司不得解除合同。

6. 小张为父亲投保骗保案——道德风险例外：父亲已确诊右肺腺癌仍投保并隐瞒，超两年后死亡。保险公司拒赔——骗保行为属道德风险，即使超出两年抗辩期也无效。背景：香港/欧美保险"严进宽出"，大陆"宽进严出"。

7. Macaura v Northern Assurance —— 保险利益经典：Macaura是木材公司唯一股东和债权人，以个人名义为公司木材投保火灾险。法院判无保险利益：股东只有分红权无所有权；债权人身份下木材未作为抵押品。保险利益必须是明确、合法的经济利益。

8. 雷兰德船舶鱼雷案 —— 近因原则经典："艾卡丽亚号"被德国潜艇鱼雷击中→拖至港口→港口当局令其离港→风浪中沉没。上议院判近因是鱼雷（战争行为=除外责任）而非风浪。Lord Shaw：近因应是造成损失结果最有效的因素，非时间上最近。

9. 老鼠咬管案(Hamilton v Pandorf) —— 近因判定：船上老鼠咬破管道→海水渗入→货物受损。上议院判海水进舱是近因（海上风险=承保风险）。老鼠仅制造了一个洞，海水是损失的有效主导原因，鼠害的除外条款仅适用于老鼠直接致损情形。

10. 酒后摔倒意外险案：张三投保意外险，餐馆饮酒后跌倒致四级伤残。病例记录"患者饮酒后摔伤头部"。保险公司拒赔——近因是饮酒（非意外），摔倒系饮酒导致。关键：是不是意外——饮酒是决定性因素。

11. 老王心脏病案：老王投保意外险，上街买菜与摩托车擦碰后心脏病发作死亡。保险公司拒赔——近因是心脏病（内在疾病），擦碰仅是诱因。实务中可能双方协商按比例赔偿。

12. 整容手术除外责任案：小李投保健康医疗险，美容医院整容术后面瘫。保险公司拒赔——美容手术属除外责任，不在理赔范围（因火灾毁容等必须进行的整容手术不算）。

13. 张三轿车代位追偿案：张三车被李四撞（李四主责75%）。法院判保险公司应先赔偿再代位追偿——主责75%由保险公司代位追偿，剩余25%除去交强险外由保险公司赔偿。司法解释四第10条：保险公司拒绝代位追偿不合理。

14. 4S店试驾案：林某试驾与货车相撞（全责），4S店已投保车损险和三者险。法院判保险公司赔偿车辆修复费后，可向签了《试驾协议书》的林某（被视为第三方）进行代位追偿。若没有试驾协议书，林某不会被视为第三方，保险公司无法追偿。

15. 老张医保追偿案：老张被老李骑车撞伤，治疗15万（医保支付10万+商保5万）。商业医疗险（补偿型）可追偿；根据《社会保险法》第30条，医保基金支付后也有权向第三人追偿。

第四单元 · 保险合同▶

4.1 保险合同概述

特点（六种）

  • 不要式合同：无需特定形式，口头书面均可（非要式合同）
  • 附和性合同（格式合同）：一方拟定条款，另一方只能选择是否接受（vs 协议性合同——双方充分协商后签订）
  • 有偿合同：享有权利须付出代价——对价：投保人支付保费，保险人承担风险
  • 条件合同：合同责任的履行取决于特定条件是否满足（投保人是否遵守条件+约定事故是否发生）
  • 双务合同（我国）：双方互享权利互担义务（vs 单务合同——美国视保险合同为单务合同）
  • 射幸合同(Aleatory)：一方付出代价换取不确定的机会——由事件的不确定性导致不等价交换（vs 等价交换合同Commutative）

保险法第13条：投保人提出保险要求经保险人同意承保→合同成立。第14条：合同成立后，投保人按约定缴保费，保险人按约定时间开始承担保险责任。

4.2 保险合同要素

主体

  • 当事人：投保人（民事行为能力+保险利益+缴费）+ 保险人（法定资格——依法成立的保险公司：国有独资/股份/有限/相互制）
  • 关系人：被保险人（财产或人身受保障+保险金请求权——可以一个或多个）+ 受益人（仅人身险有，保险金受领人——由被保险人或投保人指定，投保人指定须经被保险人同意）
  • 辅助人：代理人（受保险人委托，不能同时接受两家及以上委托）+ 经纪人（基于投保人利益，仅限有限责任公司）+ 公估人（评估鉴定）

客体：保险利益（不是保险标的本身——保险标的是保险利益的载体）

保险法第33条：不得为无民事行为能力人投保以死亡为给付条件的保险（父母为未成年子女投保除外，但有限额——10岁以下20万/10-18岁50万）

保险法第34条：以死亡为给付条件的合同，未经被保险人同意并认可保险金额的，合同无效；未经被保险人书面同意不得转让或质押。

合同内容（保险法第18条10项必备）：保险人名称住所；投保人被保人姓名住所+受益人姓名住所；保险标的；保险责任和责任免除；保险期间和保险责任开始时间；保险金额；保险费及支付办法；保险金赔偿或给付办法；违约责任和争议处理；订立合同年月日。

合同条款类型：基本条款（法定，保险人拟定须具备）vs 特约条款——保证条款+附加条款（效力优于基本条款）+协会条款（仅海上保险）

合同形式（效力层级从低到高）：投保单 → 暂保单（临时凭证，保单签发时自动失效）→ 保险单 → 保险凭证（小保单，法律效力相同，无保险条款）→ 批单（效力最高——后批优于前批，手写优于打印，加贴批优于正文批）

4.3 合同的订立与履行

成立 vs 生效

  • 成立：要约（投保人提出）+ 承诺（保险人同意承保）→ 承诺生效时成立
  • 生效：成立之日起生效，需要对价（缴保费）。可附条件（如激活）或附期限（如次日零时起保）
  • 有效要件：主体合格+内容合法+意思表示真实+不违反公共利益

合同变更：主体变更（标的权利转移，合同转让）；客体变更（标的种类数量变化→价值增减→保险利益变化）；内容变更（主体不变时权利义务变更→须投保人申请+保险人审核同意签发批单）

合同中止（仅适用于人寿保险）vs 终止

  • 中止条件：分期缴费+已缴首期+超60天宽限期未缴当期保费→合同效力暂停（宽限期内事故仍赔付可扣减欠缴保费）
  • 复效：2年内补交保费+利息+健康证明→效力恢复（保险法第36-37条）
  • 2年未复效：保险人有权解除合同。交足2年以上→退现金价值；未交足→扣手续费后退保费
  • 终止：自然终止（期满/履行完毕）+ 提前终止（标的灭失/合同解除）——永久不再存续

合同解除

  • 投保人：合同成立后可无条件解除（被保险人解除有条件限制——保险法司法解释三第2条：以书面形式通知撤销同意可认定为解除）
  • 保险人解除条件：未及时缴续期保费；标的危险程度增加；违反最大诚信原则；引发道德风险（保险法第27条——谎报事故不赔不退；故意制造事故不赔不退；伪造变造证据虚报部分不赔）

4.4 争议处理

解释原则：文义解释 → 意图解释 → 专业解释 → 有利被保险人和受益人解释（保险法第30条）

争议方式：协商 → 调解（民间/行政/仲裁/法院）→ 仲裁（一裁终局，有强制执行效力）→ 诉讼（二审终审，最激烈方式）

效力层级：批单正文、后批前批、手写打印、特约条款基本条款

经典案例（7个）

1. 幸福安康卡激活案：老孙花200元购买保险卡未激活，7月4日摔伤后当天激活（保单载明7月5日零时起生效）。保险公司以合同尚未生效拒赔。法院支持：合同可附条件和期限——激活=生效条件；代理人购买时已尽说明义务。启示：合同成立≠合同生效，生效条件未满足保险公司不担责。

2. 邻居代投保案（默示弃权）：甲为邻居乙投保重疾险并缴费，乙同意并电话确认。后乙患癌理赔，保险公司以投保人无保险利益拒赔。法院判保险公司败诉：乙已口头同意（被保险人同意即可视有保险利益）；保险公司订立时未提异议=默示弃权。

3. 未成年人投保限额案：父母为7岁儿子投保100万寿险——无效（10岁以下限20万）。爷爷为7岁孙子投保寿险——无效（只能由父母投保）。爷爷为7岁孙子投保住院医疗险——有效（不涉及死亡给付）。爷爷为10岁孙子投保100万寿险——无效（10-18岁限50万）。涉及死亡给付的险种（意外险/寿险）未成年人只能由父母投保且有限额！幼儿园代签意外险无效。

4. 团体意外险+工伤保险案：家电公司未缴工伤保险，员工小李工伤9级伤残。法院判：工伤保险待遇15万由公司支付；团意险10万另赔给小李——伤残保险金是给付性质不受损失补偿约束，可同时获工伤待遇+意外险金。但医疗费用属补偿型不可双重获赔。

5. 重症急性胰腺炎条款案（保险法第19条）：老李投保重疾险，出院诊断胰腺囊肿但未做外科剖腹手术。保险公司以条款要求"接受外科剖腹手术治疗"拒赔。法院依据保险法第19条认定该条款无效——"免除保险人依法应承担义务或加重投保人/被保险人责任的"格式条款无效。

6. 某团骑手猝死案（保险法第30条）：骑手张三投保某团意外险（突发疾病身故60万），接单后脑出血死亡。保险公司称"配送订单非某团平台"。法院判：如认为"仅限某团订单"是免责条件须在投保时作足以引起注意的提示；保险法第30条有利被保险人解释原则适用。

7. 货车超载免责条款提示义务案：张三货车雪天超载侧翻，保险公司以超载免责拒赔。法院要求保险公司证明已尽提示义务：前端界面显著性（加粗/标红/加大字号）+强制阅读程序有效性（页面停留时间合理，不足1秒=形式阅读）+操作轨迹不可篡改性（含时间戳日志）。电子投保须满足强制停留阅读程序。

中国保险业现状（2025年数据）

关键指标

  • 保险深度 = 保费收入/GDP。我国2025年约4.37%（2024约4.36%）；全球平均7.8%，发达国家10-12%。北京7.5%→黑龙江6.9%→吉林5.6%→广东5.4%
  • 保险密度 = 人均保费。我国2025年约590美元（2024年约4400元/人）；全球平均880美元，发达国家3000-10000美元。北京16234元→上海11063元→江苏6419元

WTO对外开放进程：1992年友邦→2001年入世→取消地域限制和业务范围限制→2020年外资比例可达100%（合资寿险公司外资比例限制正式取消）

保单贷款：仅限寿险，以保单作质押向保险公司借款。实质：提前支取保单现金价值。额度≤现金价值70-80%，期限≤6个月，利率一般高于同期银行贷款利率。形式：保费自动垫交+现金贷款。

存款保险制度：2015年5月1日施行。存款保险基金管理机构管理。被保险存款含人民币+外币。最高偿付50万元，超出部分从清算财产中受偿。截至2025年末参保机构3112家，基金余额1304.352亿元。银行体系脆弱性：资产负债率极高（90%以上）+资产负债期限不匹配+挤提效应。

贷款信用保证保险：信用保险——权利人投保+权利人被保险人（义务人未按时还贷→保险公司偿还）；住房抵押贷款保证保险——义务人投保+权利人被保险人（银行通常要求强制投保）。

PCS巨灾期权（1995年芝加哥期货交易所，1999年退出市场）

  • 核心逻辑：将保险损失指数化→期权形式买卖。指数=巨灾损失累积额(超2500万)/1亿美元
  • 金融本质：牛市看涨价差(Bull Call Spread)
  • 每份合约赔付 = 200 × max(min(Pt, b) - a, 0) 美元
  • 起赔点a = 保险公司起赔额 ÷ (市场份额 × 损失相关性) ÷ 1亿
  • 退出原因：巨灾债券等其他产品更受欢迎；标准化指数与实际损失间存在"基差"

PCS期权计算例题（重点）：保险公司对冲超出7500万的2500万损失，市场份额0.5%，相关性50%。起赔点a = 7500万/(0.5%×50%)/1亿 = 300点；封顶点b = (7500+2500)万/(0.5%×50%)/1亿 = 400点。一份合约最高赔付 = 100点×200美元 = 2万美元。需买2500万/2万 = 1250份。损失320亿→PCS指数=200×(320-300)=4000美元/份 → 总行权=4000×1250=500万美元；购买成本=1250×2×200=50万美元；净补偿=450万美元

巨灾债券(Cat Bond)：SPRV发行，全额抵押消除交易对手风险。LIBOR+固定利率。类型：本金保障型/本金部分保障型/本金无保障型。目的：将巨灾风险转移至资本市场。2025年发行量达创纪录256亿美元。

流行病毒巨灾债券：世界银行2017年7月首发，3年3.2亿美元。A类（低风险）2.25亿：LIBOR+6.5%；B类（高风险）0.95亿：LIBOR+11.1%。2020年4月COVID触发总赔付1.9584亿美元。B类按死亡人数阶梯式本金赔付。

中国第一只巨灾债券：2015年7月1日境外发行，发起人中再产险，发行人SPV PandaRe，保障地震风险，5000万美元。

保单贷款ABS：2016年3月31日太平人寿首发，规模5亿元期限6个月评级AAA。后续太保/平安/新华/泰康/友邦均发行。

保险资金运用渠道：银行存款（协议存款）、政府债券、金融债券、证券投资基金、直接投资股票、企业债等。保险资金特点：规模大、稳定性强、期限长（寿险长期+财险续短为长）。

第六单元 · 财产保险▶

6.1 财产保险概述

  • 业务体系：财产损失险（火灾/运输/工程）+农业险+责任险+信用保证险
  • 特征：经营内容复杂（投保对象/承保标的/承保过程与技术/危险管理复杂）；单个保险关系不对等
  • 保险法第49条（标的转让——受让人承继权利义务，须通知保险人）、第52条（危险增加通知）、第53-54条（退保费）

6.2 火灾保险

  • 起源：1666年伦敦大火→1667年牙医巴蓬创立火灾保险
  • 企业财产险标的分类：可保（具基本可保条件）→特约可保（珠宝古玩等市价变化大/铁路桥梁等价值高危险特别→须特别约定载明保单）→不可保（土地森林证券文件违章建筑非法占有财产处于紧急状态财产农作物动物）
  • 固定资产保险金额：账面原值/出险时市场价值/出险时重置价值/协议价值
  • 流动资产保险金额：最近12个月平均账面余额或出险时账面余额（不定值保险按出险时价值）
  • 险种层级：一切险 > 综合险 > 基本险（一切险只规定除外责任不约定赔付责任；基本险≈仅火灾等→综合险附加暴雨雷电洪水地震等）
  • 赔偿计算原则：全损→赔保额但以重置价值为限；部分损失→保额≥保险价值按实际赔，保额<保险价值按比例赔。施救费用另算但有上限
  • 家财险第一危险赔偿方式：不足额投保→只要实际损失不超过保额就按实际损失全额赔偿，不看投保比例——对投保人更友好
  • 附加险：机器损坏险（与财产险具互补性）、利润损失险（营业中断险——毛利润损失=营业利润+约定维持费用）、盗抢险

6.3 运输保险——机动车辆险

2020款车险改革要点

  • 3大主险：车损险（合并全车盗抢险/自燃/涉水/玻璃单独破碎/不计免赔/无法找到第三方）+三者险（责任限额提升至10-1000万）+车上人员责任险（本质是人身险）
  • 附加险：绝对免赔率特约条款（降保费）、车轮单独损失险、车身划痕险、修理期间费用补偿险、发动机进水损坏除外特约条款、法定节假日限额翻倍险、医保外医疗费用责任险
  • 三者险赔偿：交强险先扣→剩余按责任比例（全责≤100%/主责≤70%/同等≤50%/次责≤30%/无责≤10%）。机动车vs非机动车不同比例（主责80%/同等60%/次责40%/无责10%）
  • 交强险（强制）——分项限额：有责→死亡伤残18万+医疗1.8万+财产2000=20万；无责→1.8万+1800+100。按有责/无责而非责任比例！带有一定社会责任（如醉驾车祸商业三者不赔，交强险先行垫付后追偿）
  • 判断车上人员vs第三者：事故瞬间在车内→车上人员；已甩出车外→第三者
  • 车险与其他财产险不同：部分损失不扣除保额！赔完前三次第四次仍可赔。第四次如修车费35万超保额30万→赔30万后合同终止（全损）

6.4 运输保险——货物运输险

  • FOB（离岸价）/ CNF（成本加运费）/ CIF（到岸价=货价+运费+保费）——前两种买方承担保费，后一种卖方承担保费
  • 仓至仓条款：起运地仓库至目的地仓库
  • 定值保险；可背书转让随提货单转移无需保险人同意

6.5 工程保险

  • 建筑工程险：共保交叉责任（被保险人之间相互责任事故由保险人承担）；除外责任10项
  • 安装工程险：试车、考核和保证阶段危险最大
  • 科技工程险：海洋石油（分4阶段——勘探/钻探/建设/生产）+航天（发射前/发射/在轨寿命，可单独或一并投保）+核能（政策性保险特色，核事故中被保险人负绝对责任+政府补贴）

6.6 责任保险

  • 定义：以被保险人对他人依法应负的赔偿责任为标的；设有责任限额（非保险金额）
  • 索赔发生制：追溯期（保单生效前一段）+延长报告期（保单终止后一段）
  • 事故发生制：发现期（保单终止后一段时间对保险期间内发生但未及时发现的损失索赔）
  • 长尾风险（美国石棉案）：石棉潜伏期20-50年→超100万起诉讼→超1000亿美元赔偿。保险业应对：采用索赔发生制、明确追溯期、将石棉列入除外责任、成立石棉破产信托基金
  • 主要险种：公众责任险（场所/承包人/承运人责任险）、产品责任险、雇主责任险、职业责任险（医疗/律师/建筑师/会计师）、诉讼保全责任险（诉责险）

经典案例（12个）

1. 企业财产险赔偿计算：投保时市价700万（保额700万），出险时800万，损失400万+施救费20万。不足额保险：赔款=(400+20)×700/800=367.5万

2. 银行抵押贷款保险案：银行以抵押设备（800万）为标的投保，贷款500万。银行对设备仅500万保险利益（债权）→最多投保500万。收回200万后仅余300万保险利益→发生全损只赔300万。基于损失补偿原则——以保险利益为限。

3. 火灾定义案：公司投保财产综合险，机器因线路问题起火损毁。保险公司以"仅有燃烧现象不构成火灾"拒赔。法院支持：火灾须同时满足燃烧现象+偶然意外+蔓延扩大趋势。仅有燃烧而无蔓延不属于保险意义上的火灾。但若燃烧蔓延引发其他标的损毁才算火灾。

4. 厂房租赁案：公司租用厂房（租赁期一年），期满后协商同意8月20日前搬离。8月2日员工操作不当失火，设备损失30万+厂房修理费6万。法院判保险公司需赔偿设备+厂房：保险利益延续至协商搬离日期；员工小失误非重大过失。

5. 家财险-儿子玩火案：7岁儿子独自在家用打火机点燃衣服致火灾损失2万。保险人以"家庭成员故意行为"拒赔。法院判需赔：7岁为无民事行为能力人，不具备构成"故意"的主观条件，属因疏忽导致的意外火灾事故。

6. 家财险-盗抢险案：投保盗抢险保额15000元（现金限额1500元）。被盗：数码相机5000元（赔）、珍贵邮票3000元（不赔——收藏品需特约承保）、楼道皮鞋500元（不赔——非室内财产）、现金1000元（赔1500限额内）、楼下自行车500元（不赔——室外）。共赔6000元。

7-8. 车险-货拉拉改变用途案：小李面包车按家庭自用投保后每天货拉拉接单拉货→追尾事故全责。保险公司拒赔：改变车辆用途→危险程度显著增加→未通知保险人→保险法第52条不承担赔偿责任。车上人员vs第三者判断案：车辆侧翻老张被甩出→第三者（交强险应赔）。事故瞬间在车内=车上人员；事故瞬间已出车外=第三者。小李跳车后被侧翻货车砸死→第三者（跳车后已在车外）。

9. 车险-修理厂案：张三在修理厂驾驶车辆撞坏厂房。保险公司依据"营业场所维修保养期间除外"拒赔。但本案中驾驶员是车主本人（非修理厂人员接管），法院判属车辆使用过程中，保险公司需赔。

10. 航班延误险骗保案：李四用20多个身份信息购买近900次航班延误险，获赔近300万元。利用航空服务经验挑选高延误率航班，并未实际乘坐。警方认定构成保险诈骗罪（虚构不存在的被保险对象）。争议：有人认为李四支付了对价是用经验"对赌"航班延误。请结合保险利益原则和最大诚信原则谈观点。

11. 运费险骗保：张三利用4个网店账号+5个网购账号自买自卖→频繁下单→申请退款→同意退款→申请运费险理赔→数万单操作获数十万理赔款。实际运费低于保险赔付金额赚差价。属于保险诈骗。

12. 美国石棉案——长尾风险经典。石棉潜伏期20-50年，20世纪70年代起超100万诉讼+超1000亿美元赔偿。保险业应对：改用索赔发生制+明确追溯期+列入除外责任+成立石棉破产信托基金。

第七单元 · 人身保险▶

7.1 人身保险概述

  • 定义：以人的寿命和身体为保险标的
  • 事故特点：必然性、分散性、死亡率随年龄增加但相对稳定
  • 产品特点：约定给付（非补偿——生命无价）、期限较长、保单具有储蓄性
  • 业务特点：按年均衡费率计收保费；保险人逐年计提准备金；获得长期稳定资金来源；保单调整难度大；经营管理具连续性

分类

  • 按范围：人寿/健康/意外伤害；按期限：长期（人寿）/1年期和短期（健康&意外）
  • 按投保人数：个人/联合/团体——团体险特点：选择对象是团体、不需体检、费率低、采用经验费率、使用团体保险单
  • 标准体（健体）：身体/职业/道德无明显缺陷，按标准费率承保
  • 次标准体（弱体）：死亡指数较高但仍在可保范围。承保方法：保额削减（正常费率→一定时期按比例减少给付）、年龄增加法（按增加后年龄费率收费）、增加保费（危险职业/酗酒/吸烟）、除外责任（去除部分保险责任）、限制保额（恶性疾病术后高额承保）。保单复议：重新评估复查。
  • 分红险红利来源：利差益（投资收益高于预期）、费差益（成本费用低于预期）、死差益（实际死亡率低于预期）。分配方式：领现款/累积生息/抵交保费/增额缴清保险。分红比例：可分配盈余的70%。

7.2 人身保险常见条款（12大条款——核心考点）

  1. 不可抗辩条款：合同满2年+被保险人生存+未停止缴续期保费→保险人不得以投保人未履行最大诚信义务为由主张合同无效或拒绝给付
  2. 年龄误告条款：真实年龄超过可保范围→保险人可解除（超过两年除外）。少交→补交保费及利息或按比例调整给付；多交→无息返还
  3. 宽限期条款：60天，期内事故仍赔付（可扣减欠缴保费），逾期合同效力中止
  4. 保费自动垫缴条款：宽限期后未缴费→如现金价值扣除各项欠款后余额足够→自动垫交→意义：避免非故意保单失效
  5. 复效条款：2年内补交保费+利息+健康证明（复效申请书+体检报告）→效力恢复。超过2年未达成协议→保险人有权解除合同
  6. 不丧失价值条款：退保（领取退保金→保障终止）；减额缴清（保额相应降低的同类保险）；展期保险（用现金价值购买同等保额的定期寿险，保额不变但期限缩短）；自动不丧失价值选择权
  7. 保单贷款条款：额度≤现金价值70-80%，期限≤6个月，利率一般高于同期银行贷款利率
  8. 受益人条款：可指定多人+顺序+份额；投保人指定/变更身故受益人须经被保险人同意。同时死亡→推定受益人先死。保险法第43条：投保人故意造成被保险人死亡→保险人不赔（交足2年以上退现金价值给其他权利人）；受益人故意造成被保险人死亡→该受益人丧失受益权
  9. 自杀条款：2年内自杀不赔；2年后自杀赔付
  10. 犹豫期条款：10天内退保全额退款（可扣除体检费），犹豫期后退保还要扣手续费
  11. 战争除外条款：战争/军事行动/暴乱/武装叛乱为除外责任（战争中死亡率远高于预期）
  12. 保险金给付任选条款：一次性给付现金；利息收入（存本取息）；定期收入；定额收入；终身收入

保单转让条款：不侵犯受益人利益（不可变更受益人的保单转让须受益人同意）；经保险人同意并加注；绝对转让（所有权完全转移）vs 抵押转让（部分权益担保）

7.3 传统型人寿保险

  • 定期死亡险：保费低、逆选择风险高、保单失效率高。可续保条款+可转换条款（可转终身寿险，转换时不需可保证明→保费略高）。适用：低收入人群保障；终身寿险/两全保险补充；改善信用（抵押贷款偿还保险/信用人寿保险）
  • 终身寿险：终身保障、有现金价值。分红型（可分配盈余70%）和非分红型。缴费形式：趸缴/限期缴费/终身缴费
  • 年金保险（生存保险）：按给付时间（即期/延期）；按被保险人（个人/联合及生存者/联合）；按给付期限（定期/终身/最低保证年金）；按金额是否变动（定额/变额）
  • 佟第年金(Tontine)：1653年意大利佟第向法国提出。按年龄分组→死亡成员分红分给生存成员→最后生存者获全部分红→成员全部死亡分红终止本金归发起人。引发道德风险→最终被彻底禁止
  • 两全保险：生死均给付=定期寿险+生存保险。满期双倍两全保险、养老附加定期寿险、联合两全保险

7.4 创新型人寿保险

  • 变额寿险(Variable Life)：保费固定、保额不定、投资账户无保底。死亡给付=基本保额+投资账户净额
  • 万能寿险(Universal Life)：缴费灵活、保额可调、有最低收益率保障。死亡给付=身故保险金+个人账户净额
  • 变额万能寿险：万能+变额结合，缴费灵活投资灵活→无保底收益率
  • 投资连结保险（投连险）：保障+投资账户组合，投资风险由投保人承担，与分红险区别——分红险先承诺固定回报再分配超额部分；投连险不承诺固定回报
  • 个人养老金：年缴12000元，个税递延优惠，领取时按3%缴税。开户后可购买储蓄存款/国债/商业养老保险/银行理财/公募基金

7.5 人身意外伤害险

  • 意外定义（四要素）：外来、突发、非本意、不可预见
  • 保险责任：期限内遭受意外伤害→责任期限内（多为180天）死亡或残疾→意外伤害是近因
  • 残疾认定：2014年标准10级281项，按级别给付比例不同
  • 费率按职业等级定——职业等级越低费率越低；高龄可投无健康检查
  • 与财产险比较：相同点——偶然性/非故意/补偿性/短期为主/保费一次缴清；不同点——不受损失补偿约束+可获得第三方赔偿+保险金（给付性质vs补偿性质）→财险公司和寿险公司都可经营

7.6 健康保险

  • 疾病定义：非外来原因（身体内在生理原因）→非先天性原因→非规律性生理现象
  • 特别规定：免赔额（固定金额/固定比例/日历年度免赔额/免责期间）+等待期（30/60/90天，重疾≤180天，意外伤害引起的通常无等待期）+比例给付+止损条款（自付达限额后超出费用保险人补偿）
  • 重疾险：28种标准重疾定义（2020年修订版）。甲状腺癌分级赔付：TNM分期I期以下（95%发生率）赔付30%；I期以上（5%发生率）按重疾赔付
  • 收入补偿险：ADL六项基础能力（穿衣/进食/如厕/移动/洗澡/控制大小便）≥3项失能→启动赔付。全残给付公式：部分残疾收入 = 全残给付额 × (残前收入-残后收入)/残前收入
  • 保险法第46条：人身险给付后不享有向第三者追偿权（但被保险人/受益人仍有权向第三者请求赔偿）

经典案例（12个）

1. 受益人指定案例（四种情形）：李先生以自己为被保险人投保寿险10万。(1)2个儿子同为第一顺位均生存→各5万。(2)妻子第一顺位+2个儿子次顺位，均生存→妻子得10万（次顺位在第一顺位生存时不获）。(3)妻子第一顺位+2个儿子次顺位，仅次子生存→次子得10万（第一顺位死亡+次顺位平均分配+长子死亡由次子独享）。(4)2个儿子第一顺位+妻子次顺位，妻子和次子生存→次子得10万（唯一生存的第一顺位受益人）。

2. 李某杀妻骗保案：李某与廖某相识13天领证→为廖某购买意外险450万（受益人改为李某本人）→伙同周某杀害廖某。李某被判死刑。投保人故意造成被保险人死亡→保险人不赔（保险法第43条）。廖某家人起诉保险公司索300万被驳回。

3. 终身养老保险受益人案——部分受益人丧失权益：李甲投保终身养老险保额50万，受益人妻子张乙和弟弟李丙。两年后张乙释放煤气与李甲一同死亡（张乙谋害李甲后自杀）。保险公司以"受益人故意行为致被保险人死亡"拒赔。法院判：张乙丧失受益权，但李丙作为另一受益人的权利不受影响，应获50万保险金。

4. 离婚后受益人变更案：甲和乙夫妻，乙为甲投保寿险受益人"配偶"（实际甲缴保费）→离婚→甲与丙结婚→甲死亡。保险金归丙（死亡时"配偶"是丙）。若受益人为"配偶乙"而非"配偶"，则离婚后乙不再是配偶但如合同约定为不可变更受益人且乙未同意变更则仍归乙。

5. 意外险-甲误杀乙后自杀案：甲乙各投意外险15万。打猎时甲误将乙当野猪射杀，甲内疚喝农药自杀。乙的死亡：意外险应赔（被他人误杀属外来、突发、非本意）。甲的自杀：看合同是否过2年→2年内自杀不赔。

6. 意外险-手术意外死亡案（近因判断）：小李投保意外险，住院手术治疗时发生意外死亡。关键判断：手术本身是否构成独立的介入原因（近因）。手术是疾病治疗过程→若手术意外是死亡近因则不属于外来意外，不理赔。

7. 猝死案-心源性猝死：王某借款16万后投保意外险，某晚在家昏迷抢救无效死亡，诊断为心源性猝死。保险公司拒赔——WHO定义：猝死是因自然疾病突然死亡，非外来意外事故，不属意外险责任范围。

8. 猝死案-吃馒头噎死：张三吃馒头时倒地死亡，死亡证明"猝死"，咽喉有未咽下的馒头。家人以"噎死属意外"索赔。保险公司拒赔——死亡证明为猝死（非意外窒息），且遗体已火化无法进一步鉴定。

9. 重疾险-甲状腺癌投保未告知案：张三向8家保险公司投保10份重疾险（累计644万），均对"是否已购买其他人身险"答"否"，均在免体检限额内。确诊甲状腺乳头状癌后理赔，保险公司以未如实告知解除合同。法院判断取决于投保人是否故意隐瞒——若保险人无法证明故意，超2年则不可抗辩。

10. 重疾险-隐瞒糖尿病高血压案：周某2014年体检血压血糖偏高→2015年1月购买降压药→2015年3月投保重疾险对病史询问答"否"→2015年12月因糖尿病住院发现脑膜瘤。入院记录"发现血糖高1年余"。保险公司以未如实告知解除合同并拒赔。法院支持：投保人故意隐瞒重要病史，严重影响承保决定。

11. 重疾险-3岁女童暴发性心肌炎案：3岁女童因流感诱发暴发性心肌炎，确诊到去世仅3小时。母亲为其购买重疾险，保险公司以"不符合严重心肌炎条款"为由拒绝重疾理赔仅退部分保费13800元。争议焦点：条款对"严重心肌炎"的定义是否合理——是否违反保险法第19条。

12. 宝万之争中险资运用：宝能系以二级市场收购+融资融券+资管计划增持万科股份至24.26%。2017年监管介入→资管计划延期清算→2018-2021年逐步减持→累计套现超600亿。争议焦点：宝能系杠杆率高达4倍；险资是否应介入实体企业经营引发争议。
