预备知识：连续复利

0.1 什么是连续复利？

想象你把 100 元存入银行，年利率 10%：

  • 年复利：一年后得 100 × (1+0.10) = 110 元
  • 半年复利：半年后 100×(1+0.10/2)=105，一年后 105×(1+0.10/2)=110.25
  • 每天复利：100 × (1+0.10/365)^365 = 110.52
  • 连续复利：每时每刻都在利滚利 → 100 × e^0.10 = 110.52
核心公式：A 以连续复利 r 投资 T 年 → 终值 = A × \(e^{rT}\)

0.2 为什么要用连续复利？

因为求导、积分方便。比如 \(e^{rT}\) 对 T 求导就是 r×\(e^{rT}\)，干净利落。

0.3 年计 m 次 ↔ 连续复利换算

方向公式例子
年计 m 次 → 连续复利$Rc = m × ln(1 + Rm/m)$$年利率 10% 半年复利 → Rc=2×ln(1+0.05)=9.758%$
连续复利 → 年计 m 次$Rm = m × (e^{Rc/m} − 1)$连续 10% → 半年复利 = 2×(e^0.05−1)=10.25%
直觉：连续复利的数字总是比年复利小一点。因为连续复利靠"利滚利"追上。

第一章 远期和期货的定价与估值

1.1 什么是远期合约？

两个人约定：

  • 今天签合同
  • 未来某天（到期日 T）
  • 以今天定好的价格（交割价格 K）
  • 买入/卖出某样东西（标的资产）
例子：我今天跟你签合同，3 个月后以 40.50 元/股买你的股票。不管 3 个月后市场价是 30 还是 50，我都按 40.50 买。

1.2 三个核心符号（必须分清！）

符号含义打个比方
K（交割价格）合同里写死的成交价订婚时说好的彩礼
F（远期价格）现在重新签一份同样的合同会定的价格现在市场公允的彩礼行情
f（远期合约价值）你手里的这份合同值多少钱你的婚约比市场行情赚了多少/亏了多少
关键：签合同当天，F=K，f=0。但过了一段时间，市场变了，F≠K，f≠0。

1.3 远期定价三大公式（最重要！）

定价的核心逻辑是无套利：如果有两种方式能在未来得到同样的东西，那它们现在的成本必须相同。否则就有免费午餐（套利）。

类型一：无收益资产（不付红利股票、零息债券）

公式：\[F = S \times e^{rT}\]

S=40 元，r=5%（连续复利），T=3 个月=0.25 年
→ F = 40 × \(e^{0.05×0.25}\) = 40 × e^0.0125 = 40 × 1.0126 = 40.50 元

推导逻辑（理解它，公式就忘不了）：

  • 方案 A：现在花 S=40 元买股票，持有 3 个月 → 终值 S\(e^{rT}\)=40.50
  • 方案 B：签远期合约，3 个月后以 F 元买股票
  • 无套利要求：两者终值相等，所以 F=S\(e^{rT}\)

类型二：已知现金收益的资产（附息债券）

公式：\[F = (S - I) \times e^{rT}\]

其中 I = 所有期间收益的现值之和

"持有远期合约拿不到中间的利息/红利，所以定价时要把收益扣掉。"
债券当前价格 S=900 元，1 年期远期。6 个月和 12 个月各付息 40 元。
6 个月利率 9%，12 个月利率 10%（连续复利）。
第一步：算收益的现值 I
- I = 40×\(e^{−0.09×0.5}\) + 40×\(e^{−0.10×1}\) = 38.24 + 36.16 = 74.40 元
第二步：F = (900 − 74.40) × \(e^{0.10×1}\) = 825.60 × 1.1052 = 912.39 元

类型三：已知红利率的资产（股票指数、外汇）

公式：\[F = S \times e^{(r-q)T}\]

其中 q = 连续红利率（或外汇的国外利率 rf）

"资产以速率 q 持续产生收益，相当于持有成本降低 q"
S=25 元，r=10%，q=4%，T=6 个月=0.5 年
→ F = 25 × \(e^{(0.10−0.04)×0.5}\) = 25 × e^0.03 = 25 × 1.0305 = 25.76 元

三类资产公式对照表

资产类型远期价格 F实物例子
无收益$F = Se^{rT}$不付红利股票、零息债券
已知现金收益 I$F = (S−I)e^{rT}$附息债券、付红利股票
已知红利率 q$F = Se^{(r−q)T}$股票指数、外汇

1.4 持有成本视角（Cost of Carry）

把三类公式统一成一个：

\[F = S \times e^{cT}\]

其中 c = 持有成本 = 融资成本(r) + 存储成本(u) − 收益(q)

资产持有成本 c
无收益资产$c = r$
股票指数$c = r − q$
外汇$c = r − rf$
投资性商品（有存储成本）$c = r + u$
消费性商品（有便利收益 y）c ≤ r + u − y（只能给上限）

1.5 远期合约估值（合约已经签了一段时间后）

核心公式：\[f = (F - K) \times e^{-rT}\]

"当前远期价格 F 减去当初签的交割价格 K，折现到今天"

三种资产的直接估值公式：

资产合约价值 f
无收益$f = S − Ke^{−rT}$
已知收益 I$f = S − I − Ke^{−rT}$
已知红利率 q$f = Se^{−qT} − Ke^{−rT}$
签了一年期远期合约 K=45 元买股票。6 个月后股价涨到 S=50，r=10% 不变。
- 先算当前远期价格：F = 50×\(e^{0.10×0.5}\) = 52.56
- 再算合约价值：f = (52.56−45)×\(e^{−0.10×0.5}\) = 7.56×0.9512 = 7.19 元
- 即现在平仓能赚 7.19 元。

1.6 期货价格 ≠ 远期价格（何时不同？）

大多数情况下两者几乎相同。但当利率不可预测时：
情况结果原因
标的资产价格与利率正相关期货价格 > 远期价格期货多头盈利能高利率再投资，亏损能低利率融资
标的资产价格与利率负相关期货价格 < 远期价格反之
短期合约两者接近时间短影响小

1.7 期货价格与预期未来现货价格

系统风险关系直觉
正相关（k>r）F < E(S_T)期货多头承担正系统风险，要求正回报 → 期货价格应上涨
$零相关（k=r）$$F = E(S_T)$期货多头长期利润为零
负相关（k<r）F > E(S_T)空头获利

1.8 合成工具：合约方程

核心等式：


外汇远期 = 本币贷款 + 即期买外汇 + 外汇存款

更一般地：


股票多头 = 期货多头 + 无风险债券多头

这意味着：

  • 想买股票但钱不够？→ 买期货 + 买债券
  • 有股票想变现但不卖？→ 卖期货 → 合成现金

真题 1：2021 真题（2023-2024 A 卷 Q1）— 附息债券远期定价 ★★★

题目：

已知标的资产当前价格 S=150，交割价格 K=250，息票率 5.25%（年利率），每期付息 1.25 元。付息时间：距现在 30 天、120 天、210 天。无风险连续复利年利率 r=5.25%。合约期限 250 天。
(1) 求远期价格 F。
(2) 100 天后，即期价格变为 115 元，求此时远期合约的价值 f。
(3) 若此时期货价格为 130 元，求远期合约的价值 f。

详细解析：

(1) 求远期价格 F

① 判断资产类型：付息债券 → 已知现金收益型 → 用 F=(S−I)\(e^{rT}\)

② 算收益现值 I：

\[I = 1.25e^{-0.0525\times 30/365} + 1.25e^{-0.0525\times 120/365} + 1.25e^{-0.0525\times 210/365}\]

逐项计算：

  • 第 1 项：1.25 × \(e^{−0.0525×0.0822}\) = 1.25 × 0.9957 = 1.2446
  • 第 2 项：1.25 × \(e^{−0.0525×0.3288}\) = 1.25 × 0.9829 = 1.2286
  • 第 3 项：1.25 × \(e^{−0.0525×0.5753}\) = 1.25 × 0.9703 = 1.2129

I ≈ 1.2446 + 1.2286 + 1.2129 = 3.69

③ 套公式：

\[F = (150 - 3.69)e^{0.0525 \times 250/365} = 146.31 \times 1.0366 = \boxed{151.67}\]

(2) 100 天后求合约价值 f

① 先求 100 天时剩余期限 = 250−100 = 150 天

② 算此时剩余付息的现值（只剩 120 天和 210 天那两笔，距 100 天时分别是 20 天和 110 天）：

\[I' = 1.25e^{-0.0525\times 20/365} + 1.25e^{-0.0525\times 110/365} = 1.246 + 1.231 = \textbf{2.48}\]

③ 用估值公式 f=S−I'−K\(e^{−r(T−t)}\)：

\[f = 115 - 2.48 - 151.67e^{-0.0525\times 150/365}\]

\[f = 112.52 - 151.67 \times 0.9788 = 112.52 - 148.43 = \boxed{-35.91}\]

合约价值为负 → 这份远期合约的多头亏损 35.91 元。

(3) 价格 130 元时求 f

直接用 f₁=S₁−F：

\[f = 130 - 151.67 = \boxed{-21.67}\]

真题 2：2019 真题（2021-2022 A 卷 Q1）— 远期外汇套利 ★★★

题目：

即期汇率 0.6500 美元/澳元，28 天远期汇率为 0.6600 美元/澳元。澳大利亚利率 3%，美国利率 8%（年利率）。判断是否存在套利机会，如有请描述套利策略并计算利润。

详细解析：

① 算无套利远期价格（外汇用 F=S\(e^{(r−rf)T}\)）：

\[F = 0.6500 \times e^{(0.08-0.03) \times 28/365} = 0.6500 \times e^{0.05 \times 0.0767}\]

\[F = 0.6500 \times 1.00384 = 0.6525\]

② 对比：市场远期 0.6600 > 理论远期 0.6525 → 远期被高估

③ 套利策略（远期高估 → 卖远期）：

  • 借美元 → 按即期汇率买澳元 → 存澳元 → 卖澳元远期
  • 1 澳元套利利润 = 0.6600 − 0.6525 = 0.0075 美元/澳元

真题 3：2020-2021 B 卷 — 无收益资产远期 ★★

题目：

股价当前 100 元，无风险连续复利 9.25%，6 个月远期。求远期价格。

解析：

\[F = 100 \times e^{0.0925 \times 0.5} = 100 \times 1.0473 = \boxed{104.73}\]

真题 4：2021-2022 B 卷 Q1 — 含多利率的附息债券 ★★★

题目：

债券价格 99 元，150 天远期。30 天后和 120 天后各付息 1.25 元。30 天利率 6.11%，120 天利率 6.61%，150 天利率 7.02%。

解析：

① 算 I：

\[I = 1.25e^{-0.0611\times 30/365} + 1.25e^{-0.0661\times 120/365} = 1.244 + 1.224 = 2.468\]

② F = (99−2.468)\(e^{0.0702×150/365}\) = 96.532×1.0293 = 99.36

第二章 期货的对冲策略（套期保值）

2.1 套期保值是什么？

用期货的盈利来抵消现货的亏损（反之亦然），目标是风险中性。

2.2 两种基本方向

类型何时用操作例子
空头套保（Short Hedge）将来要卖资产，怕跌卖期货石油公司 3 个月后卖油 → 卖空原油期货
多头套保（Long Hedge）将来要买资产，怕涨买期货铜加工商 5 个月后买铜 → 买入铜期货
口诀：怕跌卖空，怕涨买多

2.3 基差与基差风险

基差 = 现货价格 − 期货价格

\[b = S − F\]

到期日基差 = 0（期货价格收敛于现货价格）

基差变化对套保的影响：

套保类型基差扩大 (S涨得多或F涨得少)基差缩小
空头改善恶化
多头恶化改善
记法：空头套保者最终卖现货、平期货 → 基差扩大意味着现货价格相对于期货更贵 → 利好空头。

套保后的有效价格：\[S_2 + F_1 − F_2 = \boxed{F_1 + b_2}\]

2.4 基差风险的三个来源

  • 提前平仓：没等到期就平仓 → 基差不为零
  • 便利收益：消费性资产的便利收益波动大 → 基差更不稳定
  • 资产不匹配：套保的资产 ≠ 期货的标的资产

2.5 合约选择两原则

  • 标的资产：期货价格与被套保资产相关性越大越好
  • 交割月份：选交割月在套保到期日之后、且尽量避开交割月份（交割月价格剧烈波动）

2.6 最优套期比率（最小方差法）

\[h^* = \rho \times \frac{\sigma_S}{\sigma_F}\]

  • ρ：现货与期货价格变动的相关系数
  • σ_S：现货价格变动的标准差
  • σ_F：期货价格变动的标准差
如果期货与现货完全相关（ρ=1）且波动相当 → h*=1

2.7 股指期货套期保值（重点！）

基于 Beta 的套保数量

\[N^* = \beta \times \frac{V_P}{V_F} = \beta \times \frac{V_P}{F \times m}\]

  • β：组合相对于指数的 Beta
  • V_P：组合总价值
  • F：期货价格
  • m：合约乘数（每点多少钱）
组合 204 万，β=1.5，指数期货 F×m = 200×500=10 万/份
→ N = 1.5 × 2,040,000 / 100,000 = 30.6 → 卖空 31 份

调整组合 Beta

要把 Beta 从 β_P 调整到 β_T：

\[N = \frac{\beta_T - \beta_P}{\beta_F} \times \frac{V_P}{F \times m}\]

增加 Beta = 买期货（N>0）；降低 Beta = 卖期货（N<0）

真题 5：2019 真题（2021-2022 A 卷 Q3）— 股指期货套保 ★★★

题目：

客户持有 1000 万欧元股票组合，β=1.2。MMI 指数当前 270 点，期货每点 500 美元。求应交易的期货合约数量。

\[N = 1.2 \times 74.07 = 88.89 \approx \boxed{89 \text{ 份（卖出）}}\]

因为要保护股票多头 → 卖空期货。

2.8 合成资产头寸

核心等式：\[\text{股票多头} = \text{期货多头} + \text{无风险债券}\]

三大应用：

应用操作公式
合成指数基金买债券 + 买股指期货$N = Ve^{rT}/(mF)$
现金证券化现金 + 买期货 → 获得股票暴露同上
从股票创设现金卖期货 + 持有股票 → 获得无风险收益$N = Ve^{rT}/(mF)$

2.9 向前延展的套期保值（Rolling Hedge）

当套保期限超过所有可用的期货合约 → 用短期合约不断向前滚动。

每滚动一次就多一个基差风险来源。

真题 8：习题 2 Q3 — 股票+债券组合同步调整 ★★★★

题目：

管理 2.8 亿元组合。当前：股票 1.82 亿（β=1.08），债券 0.98 亿（D=7.2）。
目标：股票 1.54 亿（β=0.9），债券 1.26 亿（D=6.0）。
股指期货：价格 129,000 元，β_f=0.97；债券期货：价格 103,000 元，D_f=7.7，β_f=1。
现金 β=0，D=0。

详细解析：

股票调整（分两步）：

(1) 把 0.28 亿股票转为现金（β=0）：

\[N_1 = \frac{(0-1.08)}{0.97} \times \frac{28,000,000}{129,000} = -241.67 \approx \boxed{-242 \text{ 份（卖出）}}\]

(2) 把剩余 1.54 亿股票 β 从 1.08 降到 0.9：

\[N_2 = \frac{(0.90-1.08)}{0.97} \times \frac{154,000,000}{129,000} = -221.53 \approx \boxed{-222 \text{ 份（卖出）}}\]

股票合计：卖出 242+222 = 464 份。

债券调整（分两步）：

(1) 把 0.28 亿现金转为债券（D=7.2），现金 D=0：

\[N_3 = \frac{(7.20-0)}{7.70} \times \frac{28,000,000}{103,000} = 254.19 \approx \boxed{+254 \text{ 份（买入）}}\]

(2) 把 1.26 亿债券 D 从 7.2 降到 6.0：

\[N_4 = \frac{(6.00-7.20)}{7.70} \times \frac{126,000,000}{103,000} = -190.64 \approx \boxed{-191 \text{ 份（卖出）}}\]

债券合计：买入 254−191 = 63 份。

第三章 利率期货与久期

3.1 即期利率与远期利率

即期利率（Spot Rate）：从现在开始、持续 n 年的零息债券利率。

远期利率（Forward Rate）：由当前即期利率隐含的未来某时段利率。

计算（连续复利）：

\[R_F = \frac{R_2 T_2 - R_1 T_1}{T_2 - T_1}\]

1 年期即期 10%，2 年期即期 10.5% → 第 2 年远期 = (10.5%×2−10%×1)/1 = 11%
连续复利下，整个期间的等价利率 = 各段利率的简单算术平均值。

3.2 三条曲线的关系

形态由高到低排列
向上倾斜远期利率 > 零息收益率 > 附息债券收益率
向下倾斜附息债券收益率 > 零息收益率 > 远期利率

3.3 期限结构三大理论

理论核心观点
预期理论长期利率 = 预期未来短期利率
市场分割理论短中长利率互不相关
流动性偏好理论远期利率 > 预期即期利率（含风险溢价）

3.4 久期（Duration）

麦考利久期

\[D = \frac{\sum_{t=1}^n t \times PV(CF_t)}{P}\]

久期就是"平均收款时间"。零息债券的久期 = 到期期限；附息债券的久期 < 到期期限。

修正久期

\[D_{mod} = \frac{D}{1+y}\]

债券价格变动近似

\[\frac{\Delta P}{P} \approx -D_{mod} \times \Delta y\]

修正久期=4，利率上升 0.5% → 价格下降约 4×0.5%=2%

组合久期

\[D_P = \sum w_i \times D_i\]（按市值加权）

3.5 凸度（Convexity）

\[\frac{\Delta P}{P} \approx -D_{mod} \times \Delta y + \frac{1}{2} \times C \times (\Delta y)^2\]

凸度为正对投资者有利：利率涨跌都对价格有正面缓冲。

3.6 基于久期的套期保值（利率期货对冲）

\[N = \frac{D_P \times V_P}{D_F \times P_F}\]

  • D_P：债券组合久期
  • D_F：期货标的债券久期
  • P_F：期货合约价格

头寸方向：

  • 持有债券多头 + 怕利率涨（债券跌） → 卖空利率期货
  • 持有债券空头 + 怕利率跌（债券涨） → 买入利率期货

第四章 互换（Swaps）

4.1 什么是互换？

双方约定在未来交换两笔不同特征的现金流。本质上是一系列远期合约的组合。

4.2 利率互换机制

  • 固定利率 ↔ 浮动利率（LIBOR）
  • 收到浮动利率 = 多头，收到固定利率 = 空头
  • 期初不需要任何现金支付，互换价值为零

互换利率（Swap Rate）：使互换初始价值为零的固定利率。

互换价差（Swap Spread） = 互换利率 − 同期限国债收益率

互换价差反映市场参与者承受利率波动风险的意愿。

4.3 利率互换估值 — 两种方法

方法一：债券组合分解（水平分解）

\[\text{互换多头} = \text{持有浮动债券} − \text{持有固定债券}\]

\[\text{互换价值} = B_{float} - B_{fixed}\]

关键性质：浮动利率债券在期初和每个利息重置日价值 = 面值。
→ 只需计算固定债券的现值。

方法二：FRA 组合分解（纵向分解）

互换 = 一系列远期利率协议的组合。每期用远期利率估值后加总。

  • 利息：12,000×4.5% = 540 万/年；本金：12,000 万

= 360,413 / 120 − 12,334 = 3,003.4 − 12,334 = −9,330.6 万 USD

真题 14：2021-2022 A 卷 / 习题 1 附加 — 互换违约损失 ★★★

题目：

金融机构与公司签 5 年期利率互换，收固定 10%，付 6 个月 LIBOR，名义本金 1000 万美元。第 3 年末公司违约，此时所有期限利率为 8%（半年复利，连续复利等价于约 7.84%）。求金融机构的违约损失。

详细解析：

① 即期损失（第 3 年末应付差额）：

\[\text{净收入/半年} = 1000 \times 0.5 \times (10\% - 8\%) = 10 \text{ 万美元}\]

② 剩余 2 年共 4 期，每次净收入 10 万（半年的利息差额），按半年 4% 贴现：

\[PV = 9.615 + 9.245 + 8.889 + 8.547 \approx 36.3 \text{ 万美元}\]

加上即期应收差额（第 3 年末还有一笔应付）：

\[5 + 36.3 = \boxed{41.3 \text{ 万美元}}\]

第五章 期权价格特性与期权平价关系

5.1 期权是什么？

期权 = 持有者有权利但没有义务做某事。买方付期权费获得权利。
远期/期货 = 双方都有义务 → 签合约时不需要付钱。
类型持有者权利最大损失最大收益
Call 多头有权以 X 买入期权费无限（股价涨到天上去）
Call 空头有义务卖出无限期权费
Put 多头有权以 X 卖出期权费X−期权费（股价跌到零）
Put 空头有义务买入X−期权费期权费

内涵价值（Intrinsic Value）：

  • Call：max(S − X, 0) → "现在执行能赚多少"
  • Put：max(X − S, 0)

状态判断：

  • 实值（ITM）：S>X（Call）/ S
  • 平值（ATM）：S≈X → 执行不赚不亏
  • 虚值（OTM）：SX（Put）→ 不会执行

5.2 影响期权价格的六因素（必背！）

因素 ↑Call 价格Put 价格直觉解释
S（股价）↑↓Call 是"低价买"的权利 → 股价越高越值钱
X（执行价）↓↑Call：X 越低越好（买入价便宜）；Put：X 越高越好（卖出价高）
T（到期时间）↑↑时间越长，发生有利变动的机会越多
$σ（波动率）$↑↑波动越大，"意外惊喜"的可能越大
r（利率）↑↓Call 是延迟付款 → 高利率省资金成本；Put 是延迟收款 → 不利
D（红利）↓↑红利使股价下跌 → Call 受损、Put 受益
记忆技巧：除了 r 和 D 对 Call/Put 方向相反，其余四个方向一致。

5.3 期权价格上下限

看涨期权：

  • 上限：c ≤ S（期权的价格不能超过股票本身，否则直接买股票更划算）
  • 下限：c ≥ max(S − X\(e^{−rT}\), 0)

看跌期权：

  • 上限：p ≤ X（Put 的最大收益就是 X，所以价格不能超过 X）
  • 下限：p ≥ max(X\(e^{−rT}\) − S, 0)

5.4 期权平价关系（Put-Call Parity）

这是最重要的期权关系式！考了无数次！

无红利：

\[\boxed{c + Xe^{-rT} = p + S}\]

理解方式：

  • 左边 = 买一个 Call + 存一笔钱（到期刚好是 X 元）
  • 右边 = 买一个 Put + 买一股股票
  • 到期日：左边和右边的价值完全一样（都是 max(S_T, X)）
  • → 今天两者的价格必须相等，否则就有套利机会！

有红利（D = 红利现值）：

\[c + D + Xe^{-rT} = p + S\]

第六章 二叉树模型（Binomial Tree）

6.1 核心思想

将未来股价简化为两种可能：上涨(u 倍) 或 下跌(d 倍)。然后从到期日一层层往回推算期权价格。

6.2 单步二叉树

参数设定：

  • u = 上涨倍数（通常 u = \(e^{σ√Δt}\)）
  • d = 下跌倍数（通常 d = 1/u）
  • p = 风险中性上涨概率

\[p = \frac{e^{r\Delta t} - d}{u - d}\]

期权定价：

\[f = e^{-r\Delta t}[p \times f_u + (1-p) \times f_d]\]

S=20, u=1.1 → S_u=22, d=0.9 → S_d=18, r=12%, K=21
→ p = (e^0.12−0.9)/(1.1−0.9) = (1.1275−0.9)/0.2 = 0.6523
→ f_u=max(22−21,0)=1, f_d=max(18−21,0)=0
→ f = \(e^{−0.12}\)×(0.6523×1+0.3477×0) = 0.8869×0.6523 = 0.58

6.3 两步二叉树（考试常考）

步骤：

  • 算期末各节点的期权价值（f_uu, f_ud, f_dd）
  • 用单步公式从期末反推到第一步（f_u, f_d）
  • 再用单步公式从第一步反推到现在（f）

6.4 重组树 vs 非重组树

重组树非重组树
关键$u×d = d×u（上下路径可交换）$u×d ≠ d×u
N 步节点数n+1 个（计算可行）2^n 个（指数爆炸）

第七章 Black-Scholes 期权定价公式

7.1 背景：股价如何运动？

股价不是直线运动，而是像花粉在水中那样随机漂移。数学上描述为：

\[dS = \mu S dt + \sigma S dz\]

  • 第一项 μSdt：漂移项 — 股票的预期增长（趋势）
  • 第二项 σSdz：扩散项 — 围绕趋势的随机波动
这意味着：ln S_T 服从正态分布，均值为 ln S_0 + (μ−σ²/2)T，方差为 σ²T。
σ²/2 哪来的？ 来自伊藤引理：d(ln S) = (μ−σ²/2)dt + σdz。因为 ln 是凹函数，波动率会"压低"对数收益率的均值。

7.2 不确定性 ∝ √T

股票价格的不确定性与时间的平方根成正比。
时间方差标准差
1 年$σ²$$σ$
2 年$2σ²$$σ√2$
T 年$Tσ²$$σ√T$

7.3 Black-Scholes 公式（考试必背！）

欧式看涨期权：

\[\boxed{c = S_0 N(d_1) - K e^{-rT} N(d_2)}\]

欧式看跌期权（由 Put-Call Parity 可得）：

\[\boxed{p = K e^{-rT} N(-d_2) - S_0 N(-d_1)}\]

d₁ 和 d₂ 的计算：

\[\boxed{d_1 = \frac{\ln(S_0/K) + (r + \sigma^2/2)T}{\sigma\sqrt{T}}}\]

\[\boxed{d_2 = d_1 - \sigma\sqrt{T}}\]

7.4 公式的直观理解

把 Call 公式看成两部分：

\[c = \underbrace{S_0 N(d_1)}_{\text{持有股票的现价}} - \underbrace{K e^{-rT} N(d_2)}_{\text{执行价的现值}}\]

  • N(d₂) = 期权在风险中性世界中被执行的概率
  • N(d₁) = 一个比 N(d₂) 稍大的数（因为 d₁ = d₂ + σ√T），与 Delta 有关
  • 公式本质：预期收益 − 预期成本

真题 19：习题 2 Q1(a) — B-S 反推股价 ★★★

题目：

2007-7-15 买入 Banque Générale 欧式 Call：C=0.74 欧元，K=50，T=3 个月。
已知 Δ=N(d₁)=0.3426，N(d₂)=0.31728，r=4%（连续复利）。求当前股价 S。

详细解析：

由 B-S 公式：C = S·N(d₁) − K·\(e^{−rT}\)·N(d₂)

代入已知数值：

\[0.74 = S \times 0.3426 - 50 \times e^{-0.04\times 0.25} \times 0.31728\]

计算 \(e^{−0.01}\)：

\[e^{-0.01} \approx 0.99005\]

代入：

\[0.74 = 0.3426S - 50 \times 0.99005 \times 0.31728\]

\[0.74 = 0.3426S - 15.708\]

\[0.3426S = 16.448\]

\[S = \frac{16.448}{0.3426} = \boxed{48.00 \text{ 欧元}}\]

真题 20：同题(b) — 波动率变化后重新定价 ★★★

题目：

1 个月后，股价跌到 43.20 欧元，σ 上升到 40%。T 剩余 2 个月。求新期权价格。
已知：N(0.7727)=0.7802, N(0.9360)=0.8254, N(0.2273)=0.5899, N(0.0640)=0.5255

详细解析：

① 算 d₁：

\[d_1 = \frac{\ln(43.20/50) + (0.04 + 0.40^2/2) \times 2/12}{0.40\sqrt{2/12}}\]

分子：ln(0.864) + (0.04+0.08)×0.1667 = −0.1462 + 0.02 = −0.1262

分母：0.40 × 0.4082 = 0.1633

\[d_1 = -0.1262/0.1633 = \mathbf{-0.7727}\]

② d₂ = d₁ − σ√T = −0.7727 − 0.1633 = −0.9360

③ 查表：

  • N(d₁) = N(−0.7727) = 1 − N(0.7727) = 1 − 0.7802 = 0.2198
  • N(d₂) = N(−0.9360) = 1 − N(0.9360) = 1 − 0.8254 = 0.1746

④ C = 43.20 × 0.2198 − 50 × \(e^{−0.04×2/12}\) × 0.1746

\[C = 9.50 - 50 \times 0.9934 \times 0.1746 = 9.50 - 8.68 = \boxed{0.82 \text{ 欧元}}\]

股价从 48→43.20（跌 10%），时间从 3 个月→2 个月，但期权价从 0.74→0.82（涨 10.8%）！

真题 21：同题(c) — Vega 效应论述题 ★★★

题目：

解释为什么股价下跌 10%、时间流逝 1 个月，但期权价格反而上升？

解析（标准答案）：

7 月 15 日期权价 0.74 对应隐含波动率约 14%（市场平静期）。

次贷危机爆发后：

  • Delta 效应：股价从 48→43.20 → 期权价格应下降
  • Theta 效应：到期时间减少 → 时间价值减少 → 期权价格应下降
  • Vega 效应：危机带来巨大不确定性 → 波动率从 14% 飙升到 40% → 期权价格大幅上升

Vega 的影响超过了 Delta 和 Theta 的联合影响。

在危机时期，波动率（Vega）是期权定价的主导因素。

第八章 希腊字母与期权对冲

8.1 五大希腊字母速览

字母名称衡量什么对 Call 的公式
$Δ (Delta)$价格敏感度股价变 1 → 期权价变多少N(d₁)
$Γ (Gamma)$Delta 的变化率股价变 1 → Delta 变多少$n(d₁)/(Sσ√T)$
$Θ (Theta)$时间衰减过一天 → 期权价减多少负值（衰减）
$ν (Vega)$波动率敏感度$σ 涨 1% → 期权价涨多少$S√T·n(d₁)
$ρ (Rho)$利率敏感度r 涨 1% → 期权价变多少$KTe^{−rT}N(d₂)$
n(d₁) = 标准正态概率密度函数（钟形曲线的高度）

8.2 Delta（Δ）详解

  • Call：Δ ∈ (0, 1)。深度价内 → Δ≈1（股价涨 1，期权也涨 1）；深度价外 → Δ≈0
  • Put：Δ ∈ (−1, 0)。深度价内 → Δ≈−1；深度价外 → Δ≈0

Delta 对冲：

  • 卖 1 份 Call（Δ=−N(d₁) 的组合贡献） + 买 N(d₁) 股股票 → 组合 Δ=0
  • 股价涨 → Delta 升 → 需要买入更多股票来维持中性

8.3 Gamma（Γ）详解

Γ 衡量 Delta 的变化有多快。Γ 始终为正（无论是 Call 还是 Put）。
  • 平价期权（S≈K）的 Γ 最大 → Delta 变化最快 → 对冲动最频繁
  • 到期日临近 → 平价期权 Γ 急剧增大 → 对冲越来越困难

8.4 Vega（ν）详解

波动率每涨 1 个百分点，期权价格涨 ν 元。ν 始终为正。
  • 平价期权 ν 最大
  • 长期期权 ν > 短期期权（时间越长，波动率影响越大）

8.5 Delta-Gamma 中性

只用股票只能消灭 Delta。要同时消灭 Gamma，需要第二种衍生品（不同执行价格的期权）。

联立方程组（x, y 为两种期权的头寸）：

\[\begin{cases} \Delta_1 \cdot n_1 + \Delta_2 \cdot n_2 + \Delta_{portfolio} = 0 &\text{（Delta 中性）}\\ \Gamma_1 \cdot n_1 + \Gamma_2 \cdot n_2 = 0 &\text{（Gamma 中性，组合 Γ 原为 0）}\end{cases}\]

真题 23：习题 2 Q5(4) — Delta-Gamma 中性 ★★★★

题目：

用 K=3600 Put（Δ=−0.357, Γ=0.000559）和 K=3800 Call（Δ=0.523, Γ=0.000646），
对组合（等价指数 3,232 份）进行 Delta-Gamma 中性。求期权头寸和初始成本。

详细解析：

设 x = K=3600 Put 头寸，y = K=3800 Call 头寸（正值=买入）。

Gamma 方程（组合本身没有 Gamma）：

\[0.000559x + 0.000646y = 0\]

\[\Rightarrow x = -\frac{0.000646}{0.000559}y = -1.156y\]

（x 与 y 符号相反 → 一买一卖）

Delta 方程（组合 Delta 由 3,232 份指数等价量产生，视为 +3,232）：

\[-0.357x + 0.523y + 3,232 = 0\]

代入 x = −1.156y：

\[-0.357(-1.156y) + 0.523y = -3,232\]

\[0.412y + 0.523y = -3,232\]

\[0.935y = -3,232\]

\[y = \boxed{-3,455}\]

（负号 = 卖出 3,455 份 K=3800 Call）

\[x = -1.156 \times (-3,455) = \boxed{+3,993}\]

（正号 = 买入 3,993 份 K=3600 Put）

初始成本：

\[\text{成本} = -3,455 \times 238.20 + 3,993 \times 179.00\]

\[= -822,981 + 714,747 = \boxed{-108,234 \text{ 欧元}}\]

负成本 = 构建这个对冲组合净赚 108,234 欧元。
